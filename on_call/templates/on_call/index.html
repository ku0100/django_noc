Almost all on-call situations will be regarding down devices

    SOP
  1. Reply to the email and let NOC know you are investigating
  2. Check Slack for any communication
  3. Check Maintenance Calendar/Emails for any scheduled maintenance
  4. Verify with Work Control (x52222) for any unscheduled/scheduled maintenance
  5. Determine whether there is a UPS associated with the device using the WP-Hubs file
  6. Determine if there are any associated UPS alarm(s) in Sparky
  7. Decide whether you need to come in to investigate further
    • Determine the purpose of the switch (i.e. if it is 360-press-sw1, we know it is probably not critical and can wait until the morning)
